Single Sign On

Introducción

Se desea acceder a múltiples aplicaciones web con una sola autenticación, a su vez serán administradas por un Dashboard que simulara una sola web para el usuario final.
Las peticiones a base de datos “CRUD y querys” se llevarán a cabo en otro aplicativo web con servicios restful por el puerto 8443 vía localhost.
