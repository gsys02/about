Introducción

Se desea acceder a múltiples aplicaciones web con una sola autenticación, a su vez serán administradas por una aplicación central llamada Dashboard.

Requerimientos:

  1. Java

    Sera el lenguaje del sistema

  2. Spring

    Framework para la plataforma web

  3. CAS

    Protocolo para usar SSO

  4. Servidor web

    Podemos elegir entre tomcat,glassfish,wildfly, etc.
