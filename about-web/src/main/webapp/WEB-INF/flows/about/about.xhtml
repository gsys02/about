Acerca de gnextia.com

Introducción

La finalidad del siguiente sistema es cubrir las 3 reglas primordiales

  1. Que funcione

    Para que funcione todos los subsistemas web en uno solo, y que se pueda administrar desde un dashboard de solo un acceso “login/logout” nos apoyaremos de SSO y Spring Securirty.

  2. De la mejor manera

    Java soporta poderosas herramientas ya liberadas para usarse, entre ellas esta Scala, Groovy y Gradle el cual se encargara de construir todo el proyecto para nuestro querido Jenkis.

  3. No mas allá

    Para liberar una versión sera forzoso aprobar todas sus pruebas unitarias y la auditoria de código de Jenkis y Sonar.

Requerimientos:

  1. Java

    Con la maquina virtual de java podremos usar los siguientes lemnguajes Java,Scala y Groovy

  2. Gradle

    Es el constructor de los proyectos

  3. Spring

    Framework para la plataforma web

  4. Jasig CAS

    Protocolo para usar SSO

  5. Archiva

    Nuestro repositorio de artefactos maven

  6. Sonar

    Para auditar la calidad de nuestro codigo

  7. Jenkis

    Para la integración continua del desarrollo de software

  8. Servidor web

    Podemos elegir entre tomcat,glassfish,wildfly, etc.
